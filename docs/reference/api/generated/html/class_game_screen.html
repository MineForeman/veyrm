Detailed Description

Main game screen UI component that orchestrates all game display elements.

The GameScreen class creates and manages the primary game interface, combining the map view, status information, message log, and inventory display into a cohesive FTXUI component. It handles input processing, layout management, and coordinates between different UI systems.

UI Components managed:

  • Map panel (game world visualization)
  • Status panel (player stats, HP, etc.)
  • Message log panel (game events)
  • Inventory panel (items and equipment)

Detailed Description

2D coordinate point with utility functions

The Point struct represents a 2D coordinate in the game world and provides common geometric operations including distance calculations, neighbor finding, bounds checking, and operator overloads for mathematical operations.

Coordinate system:

  • X-axis: left to right (0 to width-1)
  • Y-axis: top to bottom (0 to height-1)
See also
Direction namespace for common movement vectors
Map for coordinate validation

Detailed Description

Manages player's item collection with slots and stacking.

The Inventory class handles the player's personal item storage, providing slot-based organization with automatic stacking for compatible items. It supports typical roguelike inventory operations including pickup, drop, use, and management.

Features:

  • Slot-based storage (26 slots a-z by default)
  • Automatic item stacking for compatible items
  • Type-based item searching and filtering
  • Weight system (optional for future expansion)
  • JSON serialization for save/load
  • Capacity management and overflow handling

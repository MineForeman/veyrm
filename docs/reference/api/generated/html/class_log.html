Detailed Description

Global logging utility with categorized output.

The Log class provides a comprehensive logging system for debugging and development. It supports multiple log levels, category-specific logging, and both unified and separate log files for different game systems.

Features:

  • Multiple log levels (ERROR, WARN, INFO, DEBUG, TRACE)
  • Category-specific logging (combat, AI, movement, etc.)
  • Separate log files per category for focused debugging
  • Convenience macros for easy logging
  • Runtime log level filtering

Usage:

Log::init("game.log", Log::DEBUG);
LOG_INFO("Game started");
LOG_COMBAT("Player attacks monster for 5 damage");

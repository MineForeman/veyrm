Detailed Description

Field of view calculation using symmetric shadowcasting algorithm.

The FOV class implements symmetric shadowcasting to calculate what tiles are visible from a given point on the map. This system handles line-of-sight calculations for both the player's view and AI visibility checks.

The implementation uses the recursive shadowcasting algorithm that divides the field of view into 8 octants and calculates visibility for each. This provides smooth, realistic visibility that respects opaque obstacles.

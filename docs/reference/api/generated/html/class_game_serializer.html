Detailed Description

Handles saving and loading of complete game state.

The GameSerializer manages all aspects of game persistence, including manual saves, auto-saves, save file management, and version compatibility. It serializes all game components to JSON format and provides utilities for save file organization and validation.

Features:

  • Manual save to numbered slots (1-9)
  • Auto-save with rotation (-1, -2, -3)
  • Save file browsing and management
  • Version compatibility checking
  • Complete game state serialization
  • Error handling and validation

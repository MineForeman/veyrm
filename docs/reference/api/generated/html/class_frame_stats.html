Detailed Description

Tracks and analyzes frame rate and rendering performance.

The FrameStats class monitors game performance by tracking frame rate, frame times, update times, and render times. It maintains a rolling history of the last 60 frames for calculating averages and detecting performance trends.

Performance metrics tracked:

  • FPS (frames per second)
  • Total frame time
  • Update time (game logic)
  • Render time (drawing)
  • Min/max FPS over time

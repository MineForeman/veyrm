Detailed Description

Base class for all game entities (player, monsters, items)

The Entity class provides core functionality for all game objects that exist in the world. It handles position, rendering, movement, combat interfaces, and basic properties. Derived classes (Player, Monster, Item) extend this functionality for specific entity behaviors.
